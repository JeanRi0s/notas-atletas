Descrição

Aplicação em JavaScript que recebe o nome e as notas de atletas em uma competição , calcula a média das três notas do meio (desconsiderando a maior e a menor) e exibe os resultados no console.

 Funcionalidades

Recebe uma lista de atletas com suas cinco notas.

Ordena as notas em ordem crescente.

Elimina a maior e a menor nota.

Calcula a média das três notas restantes.
